# Julius Cakes - Premium Bakery System 🎂

A state-of-the-art, fully custom web-based management system built for **Julius Cakes**, a premium bakery based in Uganda. 

## ✨ Core Features
- **Premium UI/UX:** 2024+ modern interface with Bento Grids, Glassmorphism, and smooth cubic-bezier animations.
- **Custom Cake Builder:** Multi-step form allowing customers to specify flavor, size, message, and upload reference images.
- **Order Tracking Portal:** Customers can track their order status (Pending -> Baking -> Ready -> Completed) in real-time.
- **Admin Control Center:** A customized, highly organized Django Admin panel for managing orders, inventory, and customers.
- **Analytics Dashboard:** A staff-only Bento-grid dashboard showing total revenue, pending orders, and today's baking schedule.
- **Local Optimizations:** Built for the Ugandan market with Mobile Money (MoMo) checkout UI and WhatsApp direct chat integration.

## 🛠️ Tech Stack
- **Backend:** Python, Django
- **Database:** PostgreSQL (Production), SQLite (Local)
- **Frontend:** HTML5, Tailwind CSS, Vanilla JavaScript (IntersectionObserver, Micro-interactions)
- **Deployment:** Render, Gunicorn, WhiteNoise
